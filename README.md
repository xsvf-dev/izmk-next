# IZMK - NEXT

我的世界 1.20.1 外置 PVP 客户端，专为布吉岛打造。

## 构建

直接使用 `IDEA` 导入项目。

## 运行

执行 `./gradlew runClient0` 即可。`IZMK` 将会自动加载。

## 发布

执行 `./gradlew mergeRemap` 即可。
